# Mohomed Inshaf - Full Stack Developer Portfolio 🚀

A modern, sleek, and responsive single-page developer portfolio built using pure HTML, CSS, and Vanilla JavaScript. The design features a dark theme with a beautiful **Glassmorphism** UI effect.

![Portfolio Preview](img/inshaf.png) ## ✨ Features
- **Glassmorphism UI:** Frosted glass effects on cards for a modern, premium look.
- **Interactive Accordions:** Expandable, interactive sections for Core Skills and Featured Works using pure JavaScript.
- **Responsive Design:** Custom CSS Grid and Flexbox layouts optimized for both desktop and mobile screens.
- **Creative Layering:** Z-Index layering for typography overlapping the background subject image.
- **Lightweight:** Built entirely with raw HTML, CSS, and JS. No heavy frameworks or build tools required.

## 🛠️ Technologies Used
- **HTML5** - Semantic page structure
- **CSS3** - Styling, Animations, and Glassmorphism effects
- **Vanilla JavaScript** - Logic for the interactive accordion menus
- **FontAwesome** - UI Icons
- **Google Fonts** - 'Outfit' font family
